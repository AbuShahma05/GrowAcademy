# GrowAcademy 🎓

GrowAcademy is a full-stack MERN-based online learning platform.  
It allows instructors to upload courses and students to purchase and learn online.

## 🚀 Features
- User Authentication (Student & Instructor)
- Course creation and lecture uploads
- Video-based learning system
- Dashboard for teachers & students
- Responsive UI

## 🛠 Tech Stack
**Frontend:**
- React.js
- Tailwind / CSS
- Axios

**Backend:**
- Node.js
- Express.js
- MongoDB
- JWT Authentication
